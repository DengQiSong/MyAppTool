# CommonStructure
自己用的项目公共结构块
默认采用MVP模式

App  ——Application Activity Fragment Presenter等的顶级父类

Config  ——API,常量表等

Model  ——数据层(数据库操作也放在这一层)

Module ——将界面层以功能模块分配包。

Utils    ——工具类集合

Widget  ——各个可复用View集合
